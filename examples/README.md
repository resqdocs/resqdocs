# examples

Neutrale Beispiele für lokale Entwicklung und Self-Hosting (z. B. Beispiel-Konfigurationen,
`docker-compose.example.yml`). Enthält **ausschließlich Platzhalter** — keine echten Secrets,
privaten Domains, internen IPs oder personenbezogenen Daten.

> Wird im Zuge von Deployment/Docker (Landing-Page-Phase) befüllt.
